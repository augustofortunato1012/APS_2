Sobre o Projeto

Este repositório reúne todas as atividades desenvolvidas durante a disciplina de Desenvolvimento Web.
O objetivo deste projeto é disponibilizar, por meio do GitHub Pages, os exercícios realizados ao longo das aulas, organizados de forma estruturada para facilitar a navegação e a avaliação.
O projeto contempla atividades de HTML, CSS e JavaScript, incluindo o projeto **Aparecida Nutricionista**, desenvolvido durante a disciplina.
